Chip — компонент для кликабельных закрывающихся плашек, которые могут использоваться в облаке тэгов, мультисписках и пр. Используется как составной компонент в других компонентах, например в выпадающих мульти-списках (MultipleChoiceDictionaryComponent).
Текстовое значение плашки Строчный идентификатор плашки, используется для определения скрытия нужной плашки кликом по крестику, если их представлено несколько на одной странице Производит событие, оповещающее о том, что была закрыта конкретная плашка
