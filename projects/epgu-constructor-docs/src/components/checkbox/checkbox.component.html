Checkbox — базовое представление чекбокса по умолчанию. Поддерживает свойства id, required, hidden, label и др. Используется как самостоятельный компонент, так и составной в более сложных компонентах, например в CheckboxListComponent или CheckboxCubeComponent
Уникальный строковый идентификатор чекбокса Поясняющий текст-лейбл, располагаемый рядом с чекбоксом Контрол формы для связывания значения чекбокса и форм-группы, внутри которой используется чекбокс Флаг заблокированности чекбокса, управляется опосредованно через состояние контрола control.disabled Флаг обязательности чекбокса для участия в валидационных процессах формы Флаг невидимости чекбокса, используется в редких случаях, когда чекбокс выполняет служебную функцию, не виден для пользователя и по-умолчанию всегда выбран, например передается признак isNewChild в форме добавления нового ребенка Производит событие, что чекбокс с таким-то checkboxId был нажат
