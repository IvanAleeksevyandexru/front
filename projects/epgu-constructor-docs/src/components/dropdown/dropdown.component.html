Dropdown — базовое представление чекбокса по умолчанию. Поддерживает свойства id, required, hidden, label и др. Используется как самостоятельный компонент, так и составной в более сложных компонентах, например в DropdownListComponent или DropdownCubeComponent
Массив значений элементов ListElement, доступных для выбора в списке Контрол формы для связывания значения Dropdown и форм-группы Флаг, включающий/выключающий опцию полнотекстового поиска по опциям Флаг заблокированности компонента Флаг включения/отключения опции очищения выбранной раннее опции, через нажатие крестика Флаг включения/отключения визуального представления невалидности компонента Строка-плейсхолдер, которая встречает пользователя при невыбранной опции. По умолчанию — Производит событие с выбранной в списке опцией
