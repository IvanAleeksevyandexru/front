HelperText — компонент для вывода сопроводительного текста-подсказки, оформленного по дизайн системе: серый цвет шрифта и опционально стандартный размер шрифта (по умолчанию small). Самостоятельно не используется, только в составе других компонентов, например ComponentItemComponent.
{{ content }} Размер шрифта подсказки: если пустое значение, то small или 'lg' - для увеличенной подсказки Текстовое содержимое подсказки
